# ![sigma logo](http://i.imgur.com/vW5Y3Uy.png) sigma
An error analysis calculator.

## Example
Say for example you want to calculate the time constant τ of an RC circuit:
τ = R·C

R = (100 ± 10) Ω, C = (47 ± 0.47) nF
